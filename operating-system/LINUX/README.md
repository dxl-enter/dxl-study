### 3.3 linux基本命令入门
    当前短目录：ls dir
    长格式目录：ls -l（访问权限 数量 属于用户组 大小 创建时间 名称）
    显示隐藏文件：ls -a
    创建目录：mkdir 目录名称
    复制文件：cp 复制文件 复制到aaa/复制之后文件名称
    复制目录：cp -R 复制目录 复制到aaa/复制之后目录名称
    显示用户全部目录：pwd
    删除文件命令：rm 文件名
    删除目录命令：rm -r 目录名称
